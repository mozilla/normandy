Normandy is a collection of servers, workflows, and Firefox components that enables Mozilla to remote control Firefox clients in the wild based on precise criteria.

This documentation is primarily concerned with the server side components of Normandy. For details about the Firefox components, known as the Normandy Client, see the Firefox Source Docs on the subject.
